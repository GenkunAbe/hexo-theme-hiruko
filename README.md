# Prince

A simple and light theme for [Hexo].

- [Preview](https://yiliashaw.github.io/)

## Installation

### Install

``` bash
$ git clone https://github.com/yiliashaw/hexo-theme-prince.git themes/prince
```


### Enable

Modify `theme` setting in `_config.yml` to `prince`.

### Update

``` bash
cd themes/prince
git pull
```
